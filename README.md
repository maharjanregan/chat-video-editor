# chat-video-editor

A Next.js app for building a simple “chat video” editor UI: a video player, a timeline, and a chat/messages panel.

## Tech

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS

## Getting started

```bash
npm install
npm run dev
```

By default Next runs on http://localhost:3000.

If you want a different port:

```bash
npm run dev -- --port 3005
```

## Scripts

- `npm run dev` — start dev server
- `npm run build` — production build
- `npm run start` — run production server
- `npm run lint` — lint

## Status

Early scaffold + UI shell (player + timeline + chat). Functionality will evolve.
